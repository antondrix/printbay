## Setup

Follow these setup steps to get your code and software in sync.

#### 1. Install node modules

```bash
$ npm i
```

#### 2. Create server environment file

```bash
$ cp .env.server.example .env.server
```

#### 3. Update Postman requests

1. Open Postman
2. File > Import
3. Select `postman.json`


#### 4. Seed database

```bash
$ npm run seed
```
